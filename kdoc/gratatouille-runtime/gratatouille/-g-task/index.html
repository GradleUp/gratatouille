annotation class GTask(val name: String = "", val group: String = "", val description: String = "", val pure: Boolean = true)

  name = "cook",

Parameters

name

the name of the task. If empty, defaults to the name of the function.

group

the group of the task. If empty, defaults to no group. Tasks without a group are not displayed in ./gradlew --tasks by default.

description

the description of the task. If empty, defaults to no description.

pure

whether the annotated function is pure, i.e. its outputs only depends on the inputs. Impure functions are marked as non-cacheable and never up-to-date.
